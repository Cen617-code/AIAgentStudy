### 常用命令速查

| 功能 | Claude Code命令 | 备注 |
|------|----------------|------|
| 读取文件 | `Read /path/to/file` | 不要用cat |
| 写入文件 | `Write /path/to/file` | 会覆盖 |
| 编辑文件 | `Edit /path/to/file` | 部分修改 |
| 搜索文件 | `Glob *.py` | 替代find |
| 搜索内容 | `Grep "pattern"` | 替代grep |
| 执行命令 | `Bash` | 每次会重置目录 |
| 帮助 | `/help` | 查看所有命令 |
| 退出 | `/quit` | 退出Claude Code |

### 核心规则

1. **使用绝对路径** - Bash命令后工作目录会重置
2. **Write会覆盖** - Edit更适合局部修改
3. **破坏性操作会询问** - 删除文件前会确认
4. **CLAUDE.md** - 可以配置项目规则
